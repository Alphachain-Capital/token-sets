## FART-Set

### Intro
This repository contains all code for backtesting for the [FART Set](https://set-beta.tokensets.com/set/fart).

#### Description
* `Bullish = 50 MA > 100 MA > 200 MA`
  * 85% ETH - 15% DAI
* `Bearish = 50 MA < 100 MA < 200 MA`
  * 15% DAI - 85% ETH
* `Neutral = neither Bullish nor Bearish`
  * 50% ETH - 50% DAI
